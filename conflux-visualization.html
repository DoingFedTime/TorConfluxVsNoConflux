What is Conflux?
Conflux is a multipath onion routing technique that enables the Tor network to utilize multiple circuits simultaneously for a single connection. Unlike standard Tor that sends all traffic through a single 3-relay path, Conflux splits traffic across multiple paths and combines them at the exit relay, effectively doubling throughput while maintaining the same security guarantees.
